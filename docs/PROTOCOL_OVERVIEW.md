# Quantos Protocol Overview

Quantos is a post-quantum Layer 1 blockchain designed for long-horizon security, deterministic execution, and high-throughput network architecture.

Core design areas:

- Post-quantum signature support and migration path.
- DAG-native consensus and parallel transaction propagation.
- Deterministic WASM execution model.
- Resource-based smart contract model for mainnet.
- Wallet/key-management primitives for account and transaction security.

This repository is structured for audit review of the blockchain layer only.
